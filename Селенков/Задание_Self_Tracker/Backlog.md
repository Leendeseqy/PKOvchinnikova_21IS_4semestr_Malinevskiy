## Эпики

### EPIC-1: Управление записями портфолио
**Цель**: Создание системы для учета и управления профессиональными достижениями

#### User Stories:
- **US-1.1**: Как пользователь, я хочу добавлять записи о своих достижениях (проекты, публикации, конференции и т.д.), чтобы вести учет своей профессиональной деятельности
- **US-1.2**: Как пользователь, я хочу просматривать список всех своих записей в хронологическом порядке, чтобы видеть историю своих достижений
- **US-1.3**: Как пользователь, я хочу добавлять ключевые слова к записям, чтобы классифицировать свои работы по тематикам
- **US-1.4**: Как пользователь, я хочу указывать соавторов для записей, чтобы отслеживать свое сотрудничество с другими специалистами

#### Задачи:

### EPIC-2: Анализ компетенций

#### User Stories:

- **US-2.2**: Как пользователь, я хочу видеть сводку по своим компетенциям с расчетом среднего уровня, чтобы понимать свои сильные и слабые стороны
- **US-2.3**: Как пользователь, я хочу получать рекомендации по развитию слабых компетенций, чтобы знать, какие навыки нужно улучшать
- **US-2.4**: Как пользователь, я хочу видеть предопределенный набор компетенций, чтобы не создавать их с нуля

# requirements.txt (полная версия)

```txt
# Основные зависимости для системы портфолио и компетенций
# Python 3.8 или выше

# Для работы с базой данных PostgreSQL
psycopg2-binary>=2.9.5

# Для генерации отчетов в формате Word
python-docx>=0.8.11

# Для тестирования (опционально, только для разработки)
pytest>=7.4.0
pytest-mock>=3.11.0
pytest-cov>=4.1.0

# tkinter обычно идет в комплекте с Python
# Для некоторых систем может потребоваться:
# Windows: поставляется с Python
# Linux (Ubuntu/Debian): sudo apt-get install python3-tk
# Linux (Fedora): sudo dnf install python3-tkinter
# macOS: поставляется с Python

# Стандартные библиотеки Python (уже установлены):
# datetime, json, os
```

## Инструкция по установке:

```bash
# Установка основных зависимостей
pip install -r requirements.txt

# Для разработки с тестированием
pip install -r requirements.txt pytest pytest-mock pytest-cov

# Установка tkinter на Linux (если не установлен)
# Ubuntu/Debian:
sudo apt-get install python3-tk

# Fedora/RHEL:
sudo dnf install python3-tkinter
```

## Минимальные системные требования:
- Python 3.8+
- PostgreSQL 12+
- 4 ГБ ОЗУ
- 1 ГБ свободного места на диске
- Разрешение экрана 1280x720+

- macOS 11+

## Примечания:
1. Для работы приложения необходим запущенный сервер PostgreSQL
2. Рекомендуется использовать виртуальное окружение
3. Для просмотра отчетов требуется MS Word или совместимый редактор
4. Приложение не требует подключения к интернету (кроме установки зависимостей)
